# Sonic-Game-on-Scratch
Scratch Project for CS50x Week 0 Problem Set
Description:
Welcome to Sonic Lane Dodger! In this fast-paced arcade survival game, your goal is to keep Sonic alive as long as possible by dodging deadly spinning blades across two different lanes.

🎮 Controls:

Up Arrow: Dash to the Top Lane.

Right Arrow: Dash Forward

Left Arrow: Dash Backwards

Down Arrow: Dash to the Bottom Lane.

Spacebar: Jump.

⚙️ How to Play:

Sonic starts with 3 lives. If you collide with a spinning blade, you take damage and lose a life. Run out of lives, and it's Game Over!

Pro Tip: Timing is everything! Quickly switching lanes grants Sonic a brief 0.5-second "invincibility shield" that allows him to safely pass through tight spots. Master your jumps and lane switches to survive!
